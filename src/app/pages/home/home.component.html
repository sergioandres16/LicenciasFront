Bienvenido al Sistema de Gestión de Licencias

Este es un sistema integral para la gestión y validación de licencias digitales. Proporciona un acceso seguro mediante autenticación JWT con usuario y contraseña. El sistema incluye funcionalidades para validar licencias por dirección MAC, registrar nuevos usuarios y gestionar el acceso a las diferentes funcionalidades del sistema. Garantiza un control preciso y transparente del proceso de validación de licencias.

🔐 Autenticación Segura

Sistema de login con JWT para garantizar la seguridad de los datos.

🔍 Validación de Licencias

Valida licencias por dirección MAC de manera rápida y eficiente.

📊 Gestión Completa

Panel de administración para gestionar usuarios y licencias.
